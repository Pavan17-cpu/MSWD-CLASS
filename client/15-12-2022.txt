Server with Nodemon
------------------------------
npm install nodemon -d  // -d means DevDependencies
Open Package.json file in server folder and paste the script below,

  "scripts":{
    "start":"node server.js",
    "dev":"nodemon --watch server.js"

Run the file using cmd "npm run dev"
